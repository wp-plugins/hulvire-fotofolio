Plugin to show simple fotofolio.

== Description ==

Plugin for displaying fotofolio with simple picture gallery on wordpress sites. 
Simple text entry and thumbnail pictures, option to add multiple images.


== Installation ==

1. Upload `hulvire-fotofolio` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place in your pages
`[huu_fotofolio]`

This plugin needs for proper function Meta Box plugin installed.
